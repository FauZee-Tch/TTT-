# Website Pengumuman Sekolah

Website untuk menampilkan pengumuman sekolah dengan panel admin.

## Fitur
- Tampilan pengumuman yang responsive
- Panel admin untuk mengelola konten
- Kategori pengumuman (Akademik, Penting, Kegiatan, Umum)
- Pencarian dan filter
- Kalender sekolah

## Cara Menggunakan
1. Buka website di: `https://[username].github.io/website-pengumuman-sekolah`
2. Untuk akses admin: 
   - Username: `admin`
   - Password: `admin123`

## Teknologi
- HTML5
- CSS3
- JavaScript (Vanilla)
- Local Storage untuk penyimpanan data
